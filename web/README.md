# lleme-web

A static chat interface for [lleme](https://github.com/nchapman/lleme), built with [assistant-ui](https://github.com/Yonom/assistant-ui).

## Prerequisites

- [lleme](https://github.com/nchapman/lleme) server running on `localhost:11313`

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
pnpm build
```

Static files are output to `dist/`. Preview locally:

```bash
pnpm preview
```
